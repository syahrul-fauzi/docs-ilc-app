## ðŸ“… 2. Timeline Peluncuran

### Phase 1: Soft Launch (Bulan 1)
*Fokus: Validasi Teknis & Feedback Awal*
- **Target**: 1,000 User Terdaftar.
- **Channel**: Undangan terbatas ke komunitas hukum universitas & LinkedIn group.
- **Aktivitas**:
  - Rilis Beta di TestFlight/Play Store Testing.
  - Kompetisi "Bug Bounty" kecil-kecilan.
  - Webinar pengenalan fitur AI.

### Phase 2: Grand Launch (Bulan 2-3)
*Fokus: User Acquisition & Branding*
- **Target**: 10,000 User Terdaftar.
- **Channel**: Social Media Ads (IG/TikTok), PR Release, Influencer Hukum.
- **Aktivitas**:
  - Event peluncuran online/offline.
  - Diskon berlangganan "Early Bird".
  - Kemitraan dengan organisasi bantuan hukum.

### Phase 3: Growth & Retention (Bulan 4+)
*Fokus: Engagement & Monetization*
- **Target**: 50,000 User, 5% Konversi Premium.
- **Aktivitas**:
  - Fitur referral (Undang teman dapat kuota AI).
  - Newsletter hukum mingguan (AI-generated).
  - Ekspansi fitur untuk B2B (Firma Hukum).

## ðŸ“Š 4. Key Performance Indicators (KPI)

### Acquisition
- **Cost Per Install (CPI)**: Target < Rp 5,000.
- **App Store Conversion Rate**: Target > 30%.

### Activation
- **Sign-up Rate**: % user yang daftar setelah install.
- **First Query Rate**: % user yang mencoba AI Search dalam 24 jam pertama.

### Retention
- **DAU/MAU Ratio**: Stickiness produk (Target > 20%).
- **Day-30 Retention**: Target > 15%.

### Monetization
- **Conversion to Paid**: Target 2-5%.
- **Churn Rate**: Target < 5% per bulan.
